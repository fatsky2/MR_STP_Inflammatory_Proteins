Requirements.txt (R Package Dependencies)
For Mendelian Randomization (MR) Analysis + Circular Heatmap & Forest Plot Visualization
This file lists all R packages required to reproduce the analysis and visualization in this repository.
1. R Version Recommendation
R >= 4.4.0 (consistent with the original analysis environment)
2. Installation Methods
Option 1: Auto-Install via Built-in Function
Use the install_load_pkg() function in the provided R scripts to auto-install missing packages (distinguishes CRAN/Bioconductor sources automatically).
Option 2: Manual Installation

Step 1: Install CRAN Packages
install.packages(c(
  "data.table", "TwoSampleMR", "ggplot2", "writexl", "foreach",
  "ieugwasr", "tibble", "plinkbinr", "readxl", "dplyr", "circlize",
  "RColorBrewer", "dendextend", "dendsort", "gridBase", "grid", "forestploter"
), repos = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/") # Replace with your local CRAN mirror if needed

Step 2: Install Bioconductor Packages
if (!requireNamespace("BiocManager", quietly = TRUE)) install.packages("BiocManager")
BiocManager::install("ComplexHeatmap", update = FALSE)
